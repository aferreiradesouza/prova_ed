# Objetivo

Fazer um sistema de blog, e crie posts dentro de feeds.

## Requerido para funcionar

### Instalar o npm

```
npm install
```

### Instalar o Ionic

```
npm install -g ionic
```

### Iniciar o servidor

```
cd provaEd 
ionic serve
```

## Funcionalidades

* Criar feeds
* Criar posts dentro desses feeds
* Marcar ou desmarcar como lido o post

## Linguagens

* Javascript
* CSS3
* HTML5

## Frameworks

* AngularJS
* Ionic V1
